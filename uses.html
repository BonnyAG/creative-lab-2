What kind of software and equipment do I use?

Workstation

  • Apple Mac Studio (M1 Max, 64GB RAM, 2TB SSD) - 2022

    This has honestly been the best computer purchase I've ever made. It's perfect for both programming and video editing. It's main weakness however is anything that has to do with gaming.

  • Apple MacBook Air (M1, 16GB RAM, 512GB SSD) - 2020

    This is my daily driver for school as well as for any work that needs to be done out of the office. Best laptop I've ever owned.

  • Apple iPad Pro (M1, 12.9", 256 GB SSD) - 2020

    This is my go to for content, both for watching and making it because of the incredible display accuracy of this iPad.

Development Tools

  • Visual Studio Code

    For me, VS Code has become my go-to tool for anything code. The flexibility it offers with its extensions as well as the performance it offers makes it a no-brainer.

  • Arc Browser

    When it comes to doing most of my work, Arc Browser has become my browser of choice because of its performance, sidetabs, and appearance.

  • iTerm

    A great replacement for the native macOS terminal that offers more flexibility and slightly better performance.

  • Fig

    Fig improves your terminal experience by adding an autocomplete system to your terminal (similar to what VS Code does but for everywhere).

  • SQL Pro Studio & Table Plus

    A couple of great tools for managing SQL Databases using a GUI.

Design

  • Sketch

    The best design tool I've used so far. It's completely replaced Photoshop, Illustrator and Adobe XD for me.

  • Pixelmator Pro

    For any quick adjustments that need to be done on individual images.

Productivity

  • Raycast

    A more performant replacement for Spotlight on macOS that adds a ton of features to Cmd + Space (snippets, window management, calculator, unit conversion, etc.)

  • Notion

    I've found that Notion is the most effective Task Management system I've been able to use. Because of it's database capacity and multiple views, it is the most powerful and flexible option I've found.
